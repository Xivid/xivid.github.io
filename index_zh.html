研究

ChaNaTrees: Distributed Gradient Boosting Decision Tree Framework on ChaNa (Ongoing)

A distributed machine learning framework based on Gradient Boosting Decision Tree (GBDT, GBRT, GBM or MART). It is integrated as a part of ChaNa, the RDMA-optimized distributed computing engine of our group. With both system-level and algorithm-level optimizations, memory usage and communication cost are reduced, and the performance is better than state-of-the-art tools (e.g. XGBoost). Part of work submitted to SIGIR 2017.
