Selected Projects

Distributed Systems

ChaNaTrees

In Progress at Systems Research Group, Microsoft Research Asia

A distributed machine learning framework based on Gradient Boosting Decision Tree (GBDT, GBRT, GBM or MART). It is integrated as a part of ChaNa, the RDMA-optimized distributed computing engine of our group. With both system-level and algorithm-level optimizations, memory usage and communication cost are reduced, and the performance is better than popular similar tools (e.g. XGBoost).

I have been independently building it on ChaNa.

Embedded Systems

Human Body Tracking on DJI M100 UAV

Best Project of Embedded System Principle and Practice, Harbin Institute of Technology

We deploy Raspberry Pi as the "brain" of the UAV to make it follow a designated human body with the help of realtime video stream tracked from 8 cameras installed at the 4 sides. I am capable of controlling the drone's motion with Raspberry Pi over UART.

This course project later became an National Innovation Project in HIT. It is still in progress with some new members.

Programming Languages and Compilers

C6 Compiler

COMP3235 Compiling Techniques, The University of Hong Kong.

Completed in joy with my friend Sha Li (Zoey) from Shanghai Jiaotong University. Features include:

  • integer,char and static string types
  • local and global variables
  • multi-dimensional arrays
  • real strings (char arrays) and array of static strings
  • function calls (passing arguments by value or by reference)
  • input/output with format string or variable argument list (basic types and arrays)
With an assembly simulator, I also developed two executable applications for this simple language. One is a plotting demo piping into a Python script calling pyplot. Another is a small 3-puzzle game connected with an OpenGL GUI written in C. (See the image at left. That's it! üòÅ)

SINH Interpreter in Haskell

COMP3259 Principles of Programming Languages, The University of Hong Kong

SINH is a simple functional script language introduced in the course lectures. It supports first-class functions, recursion and several types of error checking.
It also has another version supporting imperative features like mutable variables.
We design and develop interpreters for both functional and imperative SINH in the functional programming language Haskell.

Computer Architecture

Single-cycle Four-beat CPU with Multiple Interrupt on FPGA

Summer Project, Harbin Institute of Technology

Flash memory layout: reading instructions in code segment, and manipulating data in data segment. An Intel-8259-like Multiple Interrupt Controller as a peripheral. PmodLCD displaying interrupt level, register contents and each instruction being executed. Controlling execution beats with a debounced button.
