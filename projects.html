Distributed Systems

ChaNaTrees

In Progress at Systems Research Group, Microsoft Research Asia

A distributed machine learning framework based on Gradient Boosting Decision Tree (GBDT, GBRT, GBM or MART). It is integrated as a part of ChaNa, the RDMA-optimized distributed computing engine of our group. With both system-level and algorithm-level optimizations, memory usage and communication cost are reduced, and the performance is better than popular similar tools (e.g. XGBoost).

I have been individually building it on ChaNa.

SINH Interpreter in Haskell

COMP3259 Principles of Programming Languages, The University of Hong Kong

We design and develop interpreters for both functional and imperative SINH in the functional programming language Haskell.

View Project

SINH is a simple functional script language introduced in the course lectures. It supports first-class functions, recursion and several types of error checking. It also has another version supporting imperative features like mutable variables.

Features:

  • First-Class Functions
  • Recursion
  • Error Checking (e.g. static type checking, division-by-0 checking)

For the imperative version, we implement mutable-state variables with simple memory management using Monads.

Computer Architecture

Single-cycle Four-beat CPU with Multiple Interrupt on FPGA

Summer Project, Harbin Institute of Technology

Written in VHDL on Xilinx ISE. Adapted to the Digilent Nexys 3 FPGA board. Flash memory layout: reading instructions in code segment, and manipulating data in data segment. Includes an Intel-8259-like Multiple Interrupt Controller as a peripheral. PmodLCD displaying interrupt level, register contents and each instruction being executed. Controlling execution beats with a debounced button.

Embedded Systems

Human Body Tracking on DJI M100 UAV

Starred Project of Embedded System Principle and Practice, Harbin Institute of Technology

We install Raspberry Pi as the "brain" of the UAV to control its flight. With the help of the realtime video stream tracked from 8 cameras deployed at 4 sides, we make the drone follow a designated human body. I am responsible for sending accurate motion controlling instructions with Raspberry Pi over UART.

This course project later became an National Innovation Project in HIT. It is still in progress with some new members.

Internet and Web Services

Bitcoin Forum Spider and Data Analysis

IBM Technology Center (FoOTOo Lab), Harbin Institute of Technology

BitcointalkSpider is a web crawler targeting at crawling down all the posts and user data of Bitcoin Forum. It is a automatically scheduled incremental spider based on the Python Scrapy framework. We came up some interesting statistics from this data with PyLab, and found the positive correlation between user activeness on Bitcoin Forum and Bitcoin price.

Developed in honor with my friend Hailong Tang.

Operating Systems

Linux Quick Launcher

IBM Technology Center (FoOTOo Lab), Harbin Institute of Technology

Linux quick launcher is a Spotlight-like application searcher and launcher for Linux desktop. All executables (CLI commands, desktop applications) can be indexed and launched by Linux Quick Launcher in different manners. It supports wildcard searching and special commands. We specially designed the result ranking algorithm to present the most likely item at the top. We implemented both commandline and PyQt GUI version.

Developed in delight with my friend Jiaqi (Nil) Wang.

RhythmHit Music Game

Best Project of XNA Based Game Design & Development, Harbin Institute of Technology

RhythmHit is a game where you drive your vehicle to hit the hazards with the rhythm beats. All hazards are designed to appear in such a time that makes the collision happens at the very time a beat in the music happens. So it will trigger the sound effect of a percussion instrument (the type of instrument varies with songs) at the same time. The game was developed in delight with my friend Jiaqi (Nil) Wang.

Drive the rocket, and catch the beat!
